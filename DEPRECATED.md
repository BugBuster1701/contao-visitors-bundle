# Deprecated features

## Insert-Tag "count"
Have been deprecated in Version 1.7 and will no longer work in Version 2.0.<br>
Use the Front end module and from version 2.0 the new content element instead.

## Front end Modul
Have been deprecated in Version 1.17 and will no longer work in Version 2.x. for Contao 6.<br>
From version 2.0.0 onwards, there is a content element as a replacement parallel to the Front end module.
